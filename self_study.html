R as a Communication Tool

R has many tools to facilitate written communication. You can write PDF articles, author presentation slides, and create websites. This website as well as all the slides for FM 5990 were created in R Studio.

Rmarkdown is the foundation of R’s vast capabilities as a communication tool. It is an extension of markdown, which is a simple markup language.
